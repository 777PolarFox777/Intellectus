----------
type of request: GET
```
*** RETURNS ***
{
	"token": "string",
	"questions": [
		{
			"problemFields": [
				"svg or link",
				"svg or link",
				...
				"svg or link"
			],
			"options": [
				"svg or link",
				"svg or link",
				...
				"svg or link"
			]
		},
		...
	]
} 
```
answers/     
-------
type of request: POST   
> Condition: only one time per token
```
*** PARAMS ***
{
	"token": "string",
	"answers": [
		0,
		3,
		...
		1
	]
}
```
```
*** RETURNS ***
{
	"solutions": [
		0,
		2,
		...
		2
	],
	"pointsDistribution": [
		15,
		26,
		...
		17
	]
}
```
